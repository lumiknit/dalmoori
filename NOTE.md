## LPG

Lack of Parser and code Generator
 : parsing/code generation helper library

- `lp`: Parser helper
- `lg`: code Generator helper
- `lpath`: Path helper
- `ltup`: Lua named-tuple helper

## LNG

Language which is Not Good
 : Simple PL compiled into Lua
